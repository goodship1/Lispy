# Operations 

```python 
#Binary Operations

(+ 1 1)#add
(- 1 10)#minus
(* 1 1)# times

#Compare Operations

(< 20 100)#Lessthan 
(> 1 0)#Greaterthan 
(<= 20 1)#Lessthanequal
(>= 2 1)# Greaterthanequal

# Define Variables 
(define x + 1 1)
(define c * 20 1)
(define y <  1 100)
(define top  if > 2  1  + 1 2 * 3)
(fac 10)# factorial 

# lists 
(list 1) # [1]
(list  1 23 1 .. n)# list of n elements
(define x list  1 2 3 1)# defining  a list variable

# Trig Operations

(cos 2) # cos function
(sin 200) sin function

# If Then  Else Statements

(if > 1 0 + 2 1  * 6 1)
(if > x y + x y * x y)

```
