# Lispy
An small Lispy Interperter  inspired  from the blog post: http://norvig.com/lispy.html


# REPL IT
Using the following command python3 Repl.py.

# Operations 

```python 
#Binary Operations

(+ 1 1)#add
(- 1 10)#minus
(* 1 1)# times

#Compare Operations

(< 20 100)#Lessthan 
(> 1 0)#Greaterthan 
(<= 20 1)#Lessthanequal
(>= 2 1)# Greaterthanequal

# Define Variables 
(define x + 1 1)
(define c * 20 1)
(define y <  1 100)
(define top  if > 2  1  + 1 2 * 3 1)



# Trig Operations

(cos 2) # cos function
(sin 200) sin function

# If Then  Else Statements

(if > 1 0 + 2 1  * 6 1)
(if > x y + x y * x y)

```
